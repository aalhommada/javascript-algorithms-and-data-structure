## Valid Anagram

```js
function isAnagram(string1, string2) {
  if (string1.length !== string2.length) {
    return false;
  }
  let sortString1 = string1.split("").sort().join("");
  let sortString2 = string2.split("").sort().join("");
  return sortString1 === sortString2;
}
```

This solution has a time complexity of **O(n log n)**, where n is the length of the strings, because the sort function has a time complexity of O(n log n). The space complexity is **O(n)** because we need to store the sorted forms of both strings in memory.
